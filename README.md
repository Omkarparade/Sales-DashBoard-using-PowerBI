# Sales Analysis Dashboard (Power BI)

## Objective
To analyze sales data and build an interactive Power BI dashboard that provides insights into sales, profit, customer segments, and regional performance.

## Tools & Technologies
- Power BI
- DAX
- Power Query
- Excel / CSV

## Key Features
- KPI cards for Sales, Profit, Quantity, Customers, and Products
- Interactive slicers (Year, Month, Region, Category)
- DAX measures for dynamic calculations
- Clean and business-focused dashboard design

## Dataset
Sample sales dataset used for analysis.
